# 💻 Portfólio Pessoal - [seu nome]

Este é o repositório do meu primeiro portfólio pessoal, desenvolvido para centralizar os meus estudos, projetos e evolução no desenvolvimento **Frontend**. O objetivo deste projeto é demonstrar de forma clara a minha progressão e as tecnologias que estou a aprender.

## 🚀 Sobre o Projeto

O site foi construído com foco em **minimalismo e funcionalidade**, utilizando uma paleta de cores verde suave para transmitir clareza. Ele serve como um cartão de visitas digital, apresentando as minhas habilidades e os primeiros projetos que desenvolvi durante a minha jornada como programadora iniciante.

### 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização personalizada, uso de Flexbox para layouts e variáveis de cor.
* **Font Awesome:** Ícones para redes sociais e categorias de tecnologias.
* **Google Fonts:** Tipografia focada na legibilidade.

## 📁 Estrutura do Site

O portfólio está dividido em 5 secções principais:
1.  **Home:** Uma breve introdução sobre quem sou e o meu foco em interfaces limpas.
2.  **Habilidades:** Barras de progresso visuais indicando o meu nível atual em HTML (80%), CSS (70%) e JavaScript (50%).
3.  **Ideias (Projetos):** Galeria de projetos desenvolvidos:
    * **Calculadora Moderna:** Operações básicas com design intuitivo.
    * **Lista de Tarefas:** Widget para gerenciar notas e tarefas com salvamento local.
    * **Widget de Clima:** Integração com API externa para busca de clima por cidade.
4.  **Sobre:** Detalhes sobre a minha transição para a área da tecnologia.
5.  **Contato:** Formulário de mensagem e links para LinkedIn, GitHub e Instagram.

## 🎨 Design e Layout

O design foi planeado para oferecer uma experiência agradável:
* **Layout Centralizado:** Largura fixa de 900px para melhor foco na leitura.
* **Cards:** Uso de sombras e transições para destacar os projetos.
* **Fundo Dinâmico:** Gradiente linear fixo para uma estética moderna.
