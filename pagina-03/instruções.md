# 🛠️ Instruções Finais para Publicar o teu Portfólio

Agora que o código base está pronto, segue estes passos para aceitares as minhas sugestões, colocares o site online e personalizares com os teus dados reais.

## 1. Aceitar as Alterações (Pull Request)
Como eu fiz um *fork* do teu projeto e melhorei o código, precisas de aceitar essas mudanças:
1. No teu GitHub, entra no teu repositório de exercícios.
2. Clica na aba **"Pull requests"**.
3. Abre o pedido que eu enviei e clica no botão verde **"Merge pull request"**.
4. Clica em **"Confirm merge"**. Agora o teu repositório tem a versão mais atualizada.

## 2. Publicar o Site (GitHub Pages)
Para que o teu portfólio vire um link real:
1. No teu repositório, vai a **"Settings"** (Definições) > **"Pages"**.
2. Em **"Branch"**, seleciona `main` e clica em **"Save"**.
3. Aguarda um minuto e o link do teu site aparecerá no topo dessa página.

## 3. Personalizar o teu Nome (Muito Importante!)
Atualmente, o código contém nomes fictícios que usei apenas para testes. **Precisas de os trocar pelo teu nome real** abrindo o ficheiro `index.html`:

* **Na Aba do Navegador:** Altera `<title>Portfólio | [seu nome]</title>`.
* **No Logótipo:** Altera `<h1>[seu nome] M.</h1>`.
* **Na Saudação:** Altera `<h2>Olá, sou [seu nome]...</h2>`.
* **No Rodapé:** Altera `<p>[seu nome] | ...</p>`.

> **Nota:** Não te esqueças de abrir o ficheiro `LICENSE` e trocar o nome "Pricilla de Souza Teixeira" pelo teu nome também!

## 4. Atualizar o `README.md`
Depois de personalizares o código, abre o ficheiro `README.md` que criei e substitui qualquer menção a "[seu nome]" pelo teu nome real e o link correto do teu site.

## 🔗 Dica sobre o Link
Atualmente, o teu link deve ser algo como `teu-utilizador.github.io/exercicios/`. 
Se quiseres algo mais profissional, podes renomear o repositório em **Settings** de `exercicios` para `meu-portfolio` ou `teunome-dev`. O link atualizará sozinho!
